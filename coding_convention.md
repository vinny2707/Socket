# Coding Convention: 

- **Dấu ngoặc nhọn(Braces): Luôn sử dung dấu ngoặc nhọn {} cho các khối lệnh, ngay cả khi chỉ có một câu lệnh**
    - **Ví dụ:**
        if (condition) {
        doSomething();
        }   
- **Đặt tên:** Có ý nghĩa với biến, hàm, class. Với hàm sử dụng camelCase, biến sử dụng snake_case, và Class sử dụng
